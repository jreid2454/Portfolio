What You Can Expect

“Once you figure out the question, then the answer is relatively easy.” - Elon Musk.

I believe there are two key aspects of problem solving that determine the quality of, and how quick you will come to, a solution. Those being: how you approach the problem, and your ability to learn how to implement the solution.

For me, the latter is done by reasoning from first principles. This essentially means to break down a problem into its most basic elements that cannot be determined from anything else. After this you reassemble the elements from the ground up, eventually properly identifying the root of the problem. Once you do this, it is easy to find the solution. But just because you know what has to be done, doesn’t mean you know how to do it.

This brings me to my second key aspect, the ability to learn how to implement that solution. Any one in programming knows that no single person will always know all of the answers right away, especially in our world today with the constant emergence of new technologies. Instead, a much more important skill is the ability to learn, since it is inevitable that there will be a new piece of technology that will be desired to be implemented in any modern project.

Being a self taught developer, I had to master those key aspects, and I believe utilizing those two things is what has made me successful so far, and what will enable me to succeed in my future endeavors.

The Smart Grocery List

The Smart Grocery List is a React project that allows the user to label and meal, and list the ingredients for that meal. It then dynamically calculates the price of said meal based off of the ingredient cost. I learned a lot about using React.createRef() to index the individual inputs, and how to pass data from a child component to a parent component by passing a function as a prop. Feel free to use it on your next trip to the super market!

Tools: React, CSS

Resum-AI

This is a website that I run and have created, that's purpose is to utilize large language models (LLM) to create customized resumes and cover letters for users based off of their personal experience as well as a job description. I built the front end in react, and use firebase to host, as well as take care of the database via firebase firestore, which is a No-SQL database. I incorporated the openai api in order to communicate with "chat gpt", that will respond in json format, enabling me to serve the information in my own custom format.

Tools: Firebase (Hosting, Authentication, Firestore), React, OpenAI API, Stripe
